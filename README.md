# Aion's Path — Official Site

Official website and central hub for **Aion's Path**.

This site is built with plain HTML, CSS, and JavaScript so it can be hosted for free on GitHub Pages.

## Structure

```text
index.html
style.css
script.js
assets/
  screenshots/
  logo/
```

## How to publish on GitHub Pages

1. Create a public GitHub repository called `aions-path`.
2. Upload these files to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Save.

The site will be available at:

```text
https://YOUR-USERNAME.github.io/aions-path/
```
